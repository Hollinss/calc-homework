The Addition Class

This class performs the addition calculation. It is a form of abstraction as it hides the complexity of the addition operation. The input is the tuple from the calculations class created earlier. The additions class inherits from the calculations class and sums up the values in the tuple.

The get_result method exhibits polymorphism since the user calls the same calculations class which outputs different results based on which specific calculations the user calls.

Inheritance In the Calculator Program

Through inheritance, a hierarchy of classes is established where child classes inherit properties from paent classes. In our program, we practice inheritance through importing from other classes in the first lines of the program:

from calc_mod.calculations.calculation import Calculation
class Division(Calculation):

Here, within the parentheses, the "Calculation" shows the inheritance in action. Now, we can access the constructor method within the calculations class which creates the base tuple. Without this practice, we would have to recycle code which proves to be tedious and difficult to debug.
